# pika-plugin-build-web-babel

using babel for @pika/plugin-build-web


https://github.com/pikapkg/builders/pull/77


## params

### format: string

esm or cjs

### extensions: string[]

defaults to ['.js','.jsx','.ts','.tsx']

### sourcemap: boolean

whether generate sourcemap file. defaults to true
